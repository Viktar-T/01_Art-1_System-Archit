# 02_pre-submission/ — the packet that goes with the manuscript

Everything we must compose **outside** the manuscript before pressing Submit at https://susy.mdpi.com.

## Files

- `cover-letter.md` — to the editor; significance, scope fit, mandatory statements.
- `suggested-reviewers.md` — 3–5 reviewers (entered into the portal field, not the cover letter).
- `conflicts-of-interest.md` — one statement per author.
- `author-contributions-credit.md` — CRediT taxonomy lines, ready to paste into `article.qmd`.
- `funding-statement.md` — funder names checked against Crossref Funder Registry.
- `data-availability-statement.md` — chosen from MDPI's canned options.
- `graphical-abstract-brief.md` — design brief for the GA image.
- `highlights.md` — 3–5 highlight bullets (optional, helpful for promotion).

## Workflow

1. Draft each file with placeholders (`[FILL — ...]`).
2. Walk through with co-authors; resolve every placeholder.
3. Copy the final wording into `article.qmd` where MDPI puts each block in the manuscript body, AND keep the source-of-truth here so we can reuse it for revisions.
4. Mark each item complete in `../03_submission/pre-flight-checklist.md`.
