# Archiving

Open access on MDPI is durable, but redundancy is policy. Aim for:
1. One **trusted general-purpose repository** (Zenodo).
2. One **institutional repository** (employer's repo).
3. Optionally a **preprint server** entry (Preprints.org) if not done pre-submission.
