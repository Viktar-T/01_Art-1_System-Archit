# 06_post-publication/ — the long tail

The work isn't done when the DOI goes live. This phase runs for the life of the paper.

## Files

- `promotion-plan.md` — first 4 weeks: ORCID, SciProfiles, ResearchGate, social, talks, internal channels.
- `archiving.md` — long-term: Zenodo, institutional repo, preprint server (if used).
- `citation-tracker.md` — recurring (quarterly) check on Scopus / Web of Science / Google Scholar.

## Discipline

- A paper that nobody reads has the same impact as one nobody wrote. Promotion is not optional.
- Archive in at least one place outside MDPI (Zenodo or institutional repo). MDPI is reliable, but redundancy is policy, not paranoia.
- Quarterly citation check is a 10-minute task; make it a scheduled reminder, not a memory test.
