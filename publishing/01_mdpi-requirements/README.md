# 01_mdpi-requirements/ — what MDPI *Energies* requires

This folder mirrors MDPI's own author instructions for *Energies* and reduces them to actionable checklists tied to this manuscript. Treat the journal page as canonical and refresh these files when MDPI updates its rules.

**Source of truth:** https://www.mdpi.com/journal/energies/instructions
**Last synced:** 2026-05-27 (Viktar / Claude session)

## Files

- `author-guidelines-checklist.md` — line-by-line submission checklist (sections, statements, files).
- `manuscript-formatting-checks.md` — format checks that should pass before submission.
- `ethics-statements.md` — research ethics, informed consent, animal use (mark N/A if not applicable, do not omit).
- `ai-use-disclosure.md` — MDPI's GenAI disclosure rules and our exact disclosure text.
- `article-processing-charges.md` — APC awareness, funding plan, waiver/discount tracking.

## Rule of thumb

If MDPI's page contradicts anything here, MDPI wins. Edit the file, date the change, commit with `Cx-pub: refresh mdpi requirements`.
